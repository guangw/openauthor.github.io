    Default severity of the diagnostic.

    Microsoft.CodeAnalysis.LocalizableString description

    An optional longer localizable description of the diagnostic.

    Microsoft.CodeAnalysis.LocalizableString messageFormat

    A localizable format message string, which can be passed as the first argument to System.String.Format(System.String,System.Object[]) when creating the diagnostic message with this descriptor. For example, for CA1001: "Implement IDisposable on '{0}' because it creates members of the following IDisposable types: '{1}'."

    Microsoft.CodeAnalysis.LocalizableString title

    A short localizable title describing the diagnostic. For example, for CA1001: "Types that own disposable fields should be disposable".

    System.Array customTags

    Optional custom tags for the diagnostic. See Microsoft.CodeAnalysis.WellKnownDiagnosticTags for some well known tags.

    System.Boolean isEnabledByDefault

    True if the diagnostic is enabled by default.

    System.String category

    The category of the diagnostic (like Design, Naming etc.). For example, for CA1001: "Microsoft.Design".

    System.String helpLinkUri

    An optional hyperlink that provides a more detailed description regarding the diagnostic.

    System.String id

    A unique identifier for the diagnostic. For example, code analysis diagnostic ID "CA1001".
